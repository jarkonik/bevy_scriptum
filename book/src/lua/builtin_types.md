# Builtin types

bevy_scriptum provides following types that can be used in Lua:

- ```Vec3```
- ```BevyEntity```

## Vec3

### Constructor

`Vec3(x: number, y: number, z: number)`

### Properties

- `x: number`
- `y: number`
- `z: number`

### Methods

None

## BevyEntity

### Constructor

None - instances can only be acquired by using built-in `entity` global variable.

### Properties

- `index: integer`
